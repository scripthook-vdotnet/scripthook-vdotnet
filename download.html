Quick Installation Guide

1

Download the Package

Click the download button above to get the latest version of ScriptHookVDotNet.

2

Extract Files

Extract the downloaded ZIP file to a temporary location on your computer.

3

Copy to Game Directory

Copy all files to your GTA V game directory (where GTAV.exe is located).

4

Install Scripts

Place your .NET scripts in the "scripts" folder within your GTA V directory.

5

Launch & Enjoy

Start GTA V and your scripts will load automatically. Press F4 to reload scripts.
